Genre JSON vs mix JSON
  • daily_weekly_genre_pools.json → PPG-Daily & PPG-Weekly only. Each key is a pool of Plex genres (plus optional filters). Each new daily/weekly playlist randomly picks one pool; playlist titles stay “Daily/Weekly Playlist N”. Nothing creates a Plex playlist named after each pool key.
  • named_genre_mix_playlists.json → PPG-Genres only. Each key becomes one Plex playlist named {key} Mix.
  • mood_groups.json → PPG-Moods only — same “one {key} Mix playlist per entry” pattern as Genres.
The object structure is the same across these files; which script reads the file is what changes the behavior.
